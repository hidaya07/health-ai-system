# 🧠 Health AI System

A full-stack AI-powered healthcare prediction system.

## 🚀 Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- ML: Python (Flask + Scikit-learn)
- Database: MongoDB
- Real-time: Socket.io

## ✨ Features
- User Authentication (Signup/Login)
- Symptom-based disease prediction
- ML confidence scoring
- Prediction history
- Real-time doctor chat (Socket.io)
- Emergency low-confidence alerts
